# Jepretgram - A simple photo sharing app
Upload photo and share it with others!

# Server dev
Run this code on terminal in server folder
```
npm run dev
```

# Client dev
run this on terminal in client folder
```
npm run dev

// to make production files
npm run build
```

# Server Api

```
// Signin
post Users/fb/

post pics/upload/
get pics/
```
